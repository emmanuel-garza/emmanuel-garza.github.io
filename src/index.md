!!! abstract annotate "Emmanuel Garza - Software Developer & Applied Mathematician"

    ![Image title](img/emmanuel_07-26-2020.jpg){ width="40%" align=left }

    I'm a Software Developer and Manager working at <a href="https://www.divergent3d.com/" target="_blank" rel="noopener noreferrer">Divergent</a> in the additive manufacturing sector. 

    Before joining Divergent, I was a Postdoctoral Scholar at USC under a <a href="https://cifellows2020.org/2020-class/" target="_blank" rel="noopener noreferrer">Computing Innovation Fellowship</a>, working on numerical methods for the solution of electromagnetic problems. In both my postdoc at USC and graduate studies at Caltech, I worked with implementing high-performance methods for solving Maxwell's equations using integral equations, and using those methods together with shape optimization techniques to optimize photonic devices.
